Gemini Api Keys: 

There are 2 free gemini api keys available, each api key belongs to separate google projects. Reason for this is to avoid rate limiting by rotating api keys. The following are the api key names in the .env.local file:

1) GEMINI_API_KEY_1

2) GEMINI_API_KEY_2
